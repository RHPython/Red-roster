# Red Rooster Harlem Website

A high-converting, multi-page restaurant website for Red Rooster Harlem.

## Tech Stack
- **Framework:** React with Vite
- **Routing:** React Router Dom
- **Styling:** Tailwind CSS
- **Animations:** Motion (Framer Motion)
- **AI Features:** Gemini 3.1 Pro via Google AI SDK
- **Icons:** Lucide React

## Setup Instructions

1. **Environment Variables:**
   Add your Gemini API key to the `.env` file (or via your hosting provider's secrets panel):
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

2. **Installation:**
   ```bash
   npm install
   ```

3. **Development:**
   ```bash
   npm run dev
   ```

4. **Build:**
   ```bash
   npm run build
   ```

## Features
- **FAQ Chat Widget:** Floating AI assistant powered by Gemini 3.1 Pro.
- **AI Menu Descriptions:** Dynamic flavor notes and pairings generated by AI.
- **Conversion Focused:** Sticky header, prominent CTAs, and lead capture forms.
- **SEO Optimized:** Dynamic meta tags, JSON-LD structured data, and sitemap.
- **Responsive Design:** Mobile-first approach with a premium brand aesthetic.
